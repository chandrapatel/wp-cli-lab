# Demo 02 — Learning WP-CLI Commands with AI

## What This Demo Is

AI is used as a **command lookup and learning tool**. It generates WP-CLI commands from plain English descriptions and explains what each part does. The attendee then copies the command and runs it themselves in terminal.

**State this clearly before starting:**

"AI doesn't know anything about our local site — it has no access to it. It generates commands based on our description. We copy, review, and run them ourselves. Think of it as a smarter, more conversational version of `wp help`."

## DEMO: Simple Lookup

**Type in AI:**

```
Give me the WP-CLI command to list all installed plugins and their current status
```

## DEMO: With a Specific Constraint

**Type in AI:**

```
How do I reset the WordPress admin password using WP-CLI without sending a notification email to the user?
```

## DEMO: Multi-Step Workflow

**Type in AI:**

```
I've migrated my WordPress site from http://localhost:8888/mysite to https://newdomain.com.

Give me the WP-CLI commands to update all the URLs in the database. I want to preview the changes before applying them.
```

## DEMO: The Review Reminder

**Type in AI:**

```
Give me a WP-CLI command to permanently delete all posts currently in the trash
```
